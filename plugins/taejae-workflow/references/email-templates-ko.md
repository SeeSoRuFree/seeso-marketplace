# 한국어 이메일 템플릿 모음

## 1. 일반 업데이트 템플릿 (Standard)

### 사용 상황
- 일상적인 콘텐츠 업데이트
- 요청된 수정사항 완료

### 템플릿

```
제목: [태재대학교 입학처] 웹사이트 업데이트 완료 안내

안녕하세요, {담당자명} 담당자님.

요청하신 웹사이트 수정 작업이 완료되어 안내드립니다.

■ 완료된 작업

{작업 목록}

■ 확인 방법

{확인 방법 안내}

※ 이전 내용이 보이는 경우: 브라우저에서 Ctrl+F5 (Windows) 또는 Cmd+Shift+R (Mac)로 새로고침해주세요.

추가 수정이 필요하시거나 문의사항이 있으시면 연락 주시기 바랍니다.

감사합니다.

{발신자}
```

## 3. 대량 업데이트 템플릿 (Bulk)

### 사용 상황
- 여러 항목 동시 업데이트
- 정기 업데이트

### 템플릿

```
제목: [태재대학교 입학처] 웹사이트 일괄 업데이트 완료 안내 ({날짜})

안녕하세요, {담당자명} 담당자님.

요청하신 웹사이트 수정 작업이 모두 완료되어 안내드립니다.

■ 업데이트 요약

총 {N}건의 항목이 업데이트되었습니다.

| 구분 | 항목 | 상태 |
|------|------|------|
| 이미지 | {N}건 | ✓ 완료 |
| 문서 | {N}건 | ✓ 완료 |
| 텍스트 | {N}건 | ✓ 완료 |

■ 상세 내역

1. {항목 1}
   - 내용: {설명}
   - 확인: {확인 방법}

2. {항목 2}
   - 내용: {설명}
   - 확인: {확인 방법}

[추가 항목...]

■ 일괄 확인 안내

모든 변경사항을 한 번에 확인하시려면:
1. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
2. 각 페이지 방문하여 확인

문의사항이 있으시면 연락 주시기 바랍니다.

감사합니다.

{발신자}
```

## 6. 배포 예정 안내 템플릿 (Scheduled)

### 사용 상황
- 예정된 배포 사전 안내
- 점검 안내

### 템플릿

```
제목: [태재대학교 입학처] 웹사이트 업데이트 예정 안내 ({날짜})

안녕하세요, {담당자명} 담당자님.

아래와 같이 웹사이트 업데이트가 예정되어 있어 사전 안내드립니다.

■ 업데이트 일정

• 일시: {날짜} {시간}
• 예상 소요 시간: 약 {N}분
• 영향 범위: {영향 범위}

■ 예정된 변경사항

{변경사항 목록}

■ 안내사항

• 업데이트 중에도 웹사이트 접속은 가능합니다.
• 일시적으로 이전 내용이 표시될 수 있습니다.
• 업데이트 완료 후 별도 안내드리겠습니다.

문의사항이 있으시면 연락 주세요.

감사합니다.

{발신자}
```

## 공통 문구 모음

### 인사말

```
안녕하세요, {담당자명} 담당자님.
안녕하세요, 태재대학교 입학처 담당자님.
안녕하세요, {부서명} 담당자님께.
```

### 완료 안내

```
요청하신 작업이 완료되어 안내드립니다.
요청하신 수정사항이 반영되었습니다.
아래와 같이 업데이트가 완료되었습니다.
```

### 확인 요청

```
확인 부탁드립니다.
편하신 시간에 확인해주시면 감사하겠습니다.
문제가 없는지 확인 부탁드립니다.
```

### 캐시 안내

```
※ 변경사항이 보이지 않는 경우:
   - Windows: Ctrl+F5로 새로고침
   - Mac: Cmd+Shift+R로 새로고침
   - 또는 브라우저 캐시 삭제 후 재접속
```

### 문의 안내

```
문의사항이 있으시면 연락 주세요.
추가 수정이 필요하시면 말씀해주세요.
확인 중 문제가 있으시면 언제든 연락 주시기 바랍니다.
```

### 맺음말

```
감사합니다.
감사드립니다.
좋은 하루 되세요.
```

## 이메일 작성 팁

### DO (권장)

- ✅ 명확한 제목으로 내용 파악 가능하게
- ✅ 핵심 내용을 상단에 배치
- ✅ 번호/불릿으로 가독성 높이기
- ✅ 확인 방법을 구체적으로 안내
- ✅ 연락처를 명시

### DON'T (지양)

- ❌ 지나치게 긴 문장
- ❌ 기술 전문 용어 남용
- ❌ 애매한 표현 ("곧", "조만간")
- ❌ 책임 회피 표현
- ❌ 과도한 이모지 사용
