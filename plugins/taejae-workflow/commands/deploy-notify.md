# Deploy-Notify Command

배포 완료 후 고객에게 발송할 이메일 초안을 자동으로 생성하는 명령어입니다.

## 사용법

```bash
/deploy-notify                    # 기본 메일 초안 작성
/deploy-notify --template urgent  # 긴급 수정 템플릿 사용
/deploy-notify --preview          # 메일 미리보기만
```

## 스킬 참조

이 명령어를 실행하면 다음 스킬이 호출됩니다:
- `skills/taejae-deploy-notify/SKILL.md`

## 메일 템플릿

| 템플릿 | 설명 |
|--------|------|
| `standard` | 일반 업데이트용 (기본값) |
| `urgent` | 긴급 수정 통보용 |
| `bulk` | 대량 업데이트용 |
| `feature` | 새 기능 추가용 |
| `bugfix` | 버그 수정 통보용 |

## 메일 구성

생성되는 메일에는 다음 내용이 포함됩니다:

1. **완료된 작업 목록**
   - 티켓 정보 기반
   - 변경사항 요약

2. **확인 방법 안내**
   - 페이지별 확인 포인트
   - 캐시 새로고침 안내

3. **문의 안내**
   - 추가 수정 요청 방법
   - 연락처 정보

## 중요 사항

메일은 **자동 발송되지 않습니다**.

생성된 초안을 확인 후 수동으로 복사하여 발송해야 합니다.
